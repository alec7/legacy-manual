# Grunt

*Grunt* is used for developer tasks.

## Tasks

| Task             | Description                                      |
| ---------------- | ------------------------------------------------ |
| all              | Build everything                                 |
| test             | Run all tests                                    |
| mochaTest        | Run server unit tests                            |
| eslint           | Run javascript linting                           |
| csslint          | Run css linting                                  |
| validate_xml     | Run HTML linting                                 |
